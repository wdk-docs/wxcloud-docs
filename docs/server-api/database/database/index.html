数据库¶

wx.cloud.database([{env}])¶

获取数据库的引用

参数

  • env (string) – 环境 ID

返回类型

Database

示例

以下调用获取默认环境的数据库的引用：

const db = wx.cloud.database()

假设有一个环境名为 test，用做测试环境，那么可以如下获取测试环境数据库：

const testDB = wx.cloud.database({
  env: 'test'
})
