inc¶

wx.cloud.database.command.inc(value)¶

更新指令。用于指示字段自增某个值，这是个原子操作，使用这个操作指令而不是先读数据、再加、再写回的好处是：

  1. 原子性：多个用户同时写，对数据库来说都是将字段加一，不会有后来者覆写前者的情况

  2. 减少一次网络请求：不需先读再写 mul 指令同理。

参数

  • value (number) – 某个值

返回类型

Command

示例

将一个 todo 的进度自增 10：

const _ = db.command
db.collection('todos').doc('todo-id').update({
  data: {
    progress: _.inc(10)
  }
})
