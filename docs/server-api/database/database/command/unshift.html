unshift¶

wx.cloud.database.command.unshift(values)¶

更新指令，对一个值为数组的字段，往数组头部添加一个或多个值。或字段原为空，则创建该字段并设数组为传入值。

参数

  • values (any[]) – 查询条件

返回类型

Command

示例
const _ = db.command
db.collection('todos').doc('doc-id').update({
  data: {
    tags: _.unshift(['mini-program', 'cloud'])
  }
})
