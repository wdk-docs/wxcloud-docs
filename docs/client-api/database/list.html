数据库 API¶

wx.cloud.database([{[env]}])¶

获取数据库的引用

参数

  • env (string) – 环境 ID

返回

Database

示例
const db = wx.cloud.database()
wx.cloud.database.collection(name: string)¶

获取集合的引用

参数

  • name (string) – 指定需引用的集合名称

返回

Collection

示例
const collection = db.collection('todos')

小程序·云开发提供了丰富的数据库操作 API，此处是数据库小程序端的 API 参考文档。

数据库 API 都是懒执行的，这意味着只有真实需要网络请求的 API 调用才会发起网络请求， 其余如获取数据库、集合、记录的引用、在集合上构造查询条件等都是不会触发网络请求的。

获取引用API¶

API

说明

database

获取集合引用，返回 Collection 对象

获取对一个记录的引用，返回 Document 对象

集合 (Collection) API：¶

API

说明

doc

获取对一个记录的引用，返回 Document 对象

add

在集合上新增记录

构建一个在当前集合上的查询条件，返回 Query，查询条件中可使用查询指令

limit

指定返回数据的数量上限

field

指定返回结果中每条记录应包含的字段
