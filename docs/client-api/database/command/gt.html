4.2.2.5. gt¶

wx.cloud.database.command.gt(value)¶

查询筛选条件，表示字段需大于指定值。可以传入 Date 对象用于进行日期比较。

参数

  • value (number | Date) – 查询条件

返回类型

Command

示例

找出进度大于 50 的 todo

const _ = db.command
db.collection('todos').where({
  progress: _.gt(50)
})
  .get({
    success: console.log,
    fail: console.error
  })
